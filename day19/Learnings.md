new Image():
- creates a new image object
- can be used to create an image element in the DOM

'"load" and error "events"':
- are event handlers for the image object
- NOTE - not for the image element or any other element in the DOM.

Promise.all : 
- is used to wait for multiple promises to resolve concurrently before executing a function
- takes an array of promises as an argument
- resolves if all of the promises in the array resolve
- returns an array of the values of the resolved promises
- NOTE - rejects if any of the promises in the array reject
